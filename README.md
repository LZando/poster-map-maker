# Poster Map Maker

Poster Map Maker is a tool to transform your favorite cities into beautiful, minimalist designs. Create and export visually striking map posters with ease.

## Attribution

This project was inspired by and contains code adapted from the original repository:
**[originalankur/maptoposter](https://github.com/originalankur/maptoposter)**

Huge thanks to **Ankur Gupta (@originalankur)** and the contributors of `maptoposter` for their amazing work on the original city map poster generator.

## Features

- Generate beautiful, minimalist map posters for any city.
- Custom themes and styles.
- High-quality output suitable for printing.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the following command to start the application:

```bash
python run.py
```

Then follow the instructions in the interface to generate your posters.
